Gällande appen

Jag tog chansen att istället för att jämföra 2 städer, gick jag för att kunna jämföra väder mellan mans nuvarande position
och jämföra med andra städer genom att skriva rätt stads namn.

Jag tänkte lägga lite styling för att det ska kännas tillfredsställande och för att det ser bättre och finare ut med lite styling.
Jag ville poängtera att jag skapade först en autocomplete mha API:t google places för att kunna hämta rätt stads namn, om man
vill utgå nu att användaren kan skriva fel namn på staden. Men tyvärr så tog jag bort den för att responsen på den va seg och
man behövde betala för att få bättre service från den apin. Tänkte bara berätta hur jag tänkte här.

Appens egenskaper
- Kunna se vädret i sin nuvarande position om man tillåter det, annars får man meddelande att geolocation är blockerad.
- Kunna byta från C till F temperaturer om man klickar på temperaturen i första container.
- Kunna se information om vädret också, tex dimmigt, regnigt, molnigt osv.
- Kunna hämta väder för vilken stad som helst i världen.
